L'objectif de ce projet est de concevoir un site web adapté aux besoins et attentes d'une personne spécifique.

Missions

Dans le cadre de ce projet, notre première étape a été de bien comprendre notre client, en identifiant précisément ses besoins pour le site web. Une fois cette phase de réalisation, nous avons conçu les wireframes des différentes pages, puis les maquettes, afin d'avoir une vision claire du rendu final. Ensuite, nous avons développé le site web en utilisant EJS et créé notre propre base de données avec SQL. Le projet s'est conclu par une présentation orale, où nous avons pu démontrer notre compréhension et la qualité de notre travail.
